Pour utiliser ce contrôle personnalisé dans un fichier XAML dans un autre projet, procédez comme suit :

  • Ajouter une référence à ce projet.
  • Ajouter la ligne suivante à l'élément racine du fichier XAML dans lequel vous voulez utiliser ce contrôle :
    xmlns:MyNamespace="clr-namespace:EIEx_Vue;assembly=EIEx_Vue"
  • Utiliser le contrôle dans le fichier XAML :
    <MyNamespace:CustomControl1/>
